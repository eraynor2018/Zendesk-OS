# Zendesk OS

Weekly Report Generator for SidelineSwap's Zendesk support operations.

## Setup

```bash
npm install
npm run dev
```

## Supabase Setup

1. Create a project at [supabase.com](https://supabase.com)
2. Run the SQL in `supabase/schema.sql` in the SQL Editor
3. Copy `.env.example` to `.env` and fill in your Supabase URL and anon key

## Deploy

Deployed to Vercel. Push to `main` to trigger a deploy.
